# 抖音视频分镜工具 — 团队使用指南

## 工具简介

输入抖音视频链接，自动完成**下载 → 语音转文字 → 语义分镜 → 画面场景检测 → 融合对齐**，最后在网页上输出带精确时间戳的结构化分镜结果。

适用场景：短视频内容分析、二创剪辑参考、脚本拆解。

---

## 一、系统要求

| 项目 | 最低要求 | 推荐 |
|------|---------|------|
| 操作系统 | Windows 10 / 11 | Windows 11 |
| Python | 3.11 - 3.13 | 3.12 |
| 内存 | 4 GB | 8 GB（处理长视频需要更多） |
| 硬盘空间 | 2 GB 可用 | 5 GB |
| 网络 | 能访问 huggingface.co | 国内用户已配置镜像加速 |
| 显卡（可选） | NVIDIA GPU + CUDA | 加速语音转写 5-10 倍 |

## 二、安装步骤

### 第 1 步：安装 Python

如果你的电脑还没有 Python：

1. 打开 https://www.python.org/downloads/
2. 下载 **Python 3.12** 版本
3. **安装时务必勾选** "Add Python to PATH"（这是最关键的一步）
4. 安装完成后，打开命令提示符（`Win + R` → 输入 `cmd` → 回车），输入以下命令验证：

```cmd
python --version
```

如果显示 `Python 3.12.x` 则表示安装成功。

> ⚠️ 不要从 Microsoft Store 安装 Python，请从官网下载。

### 第 2 步：解压文件

1. 把收到的 `抖音视频分镜工具_v1.0.zip` 解压到 **桌面** 或 **D 盘根目录**
2. 解压后你会看到这些文件：

```
抖音视频分镜工具_v1.0/
├── app.py                    # 主程序
├── config.py                 # 配置
├── requirements.txt          # Python 依赖清单
├── core/                     # 核心逻辑
├── models/                   # 数据模型
├── utils/                    # 工具函数
├── tests/                    # 测试
├── scripts/                  # 辅助脚本
├── 首次安装.bat              # ← 先点这个
├── 启动工具.bat              # ← 以后点这个
├── 获取抖音Cookie.bat        # ← 配置 Cookie
└── 配置说明.txt              # 配置说明
```

### 第 3 步：首次安装

**双击 `首次安装.bat`**，脚本会自动完成以下操作（整个过程约 3-10 分钟，取决于网络速度）：

1. ✅ 检查 Python 版本
2. 📦 创建虚拟环境（`venv/` 目录）
3. 📦 安装所有依赖库
4. 📦 下载 ffmpeg（约 60 MB）
5. 📦 安装 Playwright 浏览器引擎（约 150 MB）
6. 📄 生成 `.env` 配置文件模板

耐心等待，期间命令提示符窗口会实时显示进度。**不要中途关闭窗口**。

如果某一步失败，脚本会提示你手动操作。

### 第 4 步：配置 API Key

安装完成后，在文件夹中找到 `.env` 文件，用 **记事本** 打开。

找到这一行：

```
DEEPSEEK_API_KEY=你的API密钥
```

把它替换成管理员给你的密钥，例如：

```
DEEPSEEK_API_KEY=sk-abcdefghijklmnopqrstuvwxyz
```

> 💡 `.env` 文件默认是隐藏文件。如果在文件夹里看不到，可以在记事本菜单「文件 → 打开」中，文件类型选「所有文件」就能看到了。

### 第 5 步：获取抖音 Cookie（可选但推荐）

如果不配置 Cookie，部分抖音视频可能无法下载（尤其是较新发布的视频）。

**双击 `获取抖音Cookie.bat`**：

1. 会自动打开一个浏览器窗口，进入抖音首页
2. 如果未登录抖音，请用 **抖音 App 扫码登录**
3. 登录成功后，脚本会自动提取 Cookie 并保存到 `.env`
4. 关闭浏览器窗口即可

> 🔄 Cookie 有时效性（通常几天到几周），过期后重新运行此脚本即可刷新。

## 三、日常使用

### 启动工具

以后每次使用，**双击 `启动工具.bat`**：

1. 等待几秒钟，命令行显示服务已启动
2. **浏览器会自动打开**，显示工具界面
3. 如果浏览器没有自动打开，手动访问 `http://127.0.0.1:7860`

### 分析单条视频

1. 在输入框粘贴抖音视频链接（支持短链接 `v.douyin.com/xxx` 和完整链接）
2. 点击「开始分析」
3. 等待处理完成（1-5 分钟，取决于视频长度）
4. 查看结构化的分镜结果：每个分镜有起止时间、文字内容、画面描述

### 批量分析

1. 切换到「批量分析」标签页
2. 粘贴多个抖音视频链接（每行一个，最多 50 条）
3. 点击「开始批量分析」
4. 每条视频会依次处理，处理完成后显示完整的分镜表格

## 五、常见问题

### Q: 语音转写太慢怎么办？

默认使用 CPU 进行语音识别。如果你有 **NVIDIA 显卡**：

1. 打开 `.env` 文件
2. 把 `WHISPER_DEVICE=cpu` 改成 `WHISPER_DEVICE=cuda`
3. 速度会提升 5-10 倍

### Q: 模型下载失败 / 网络超时

`.env` 中已默认配置了国内 HuggingFace 镜像：

```
HF_ENDPOINT=https://hf-mirror.com
```

如果不需要（例如在国外），删除这一行即可使用官方源。

### Q: 提示 "ffmpeg 未找到"

1. 检查 `bin\` 目录下是否有 `ffmpeg.exe`
2. 如果没有，重新运行 `首次安装.bat`
3. 或者手动下载 https://www.gyan.dev/ffmpeg/builds/ffmpeg-release-essentials.zip
4. 解压后把 `ffmpeg.exe` 复制到 `bin\` 目录

### Q: 视频下载失败

1. 先运行 `获取抖音Cookie.bat` 确保 Cookie 有效
2. 检查网络是否能正常访问抖音
3. 如果抖音反爬策略变更，可能需要更新下载引擎

### Q: 浏览器打开后页面空白

手动刷新页面（按 `F5` 或 `Ctrl+R`）。

### Q: 如何更新工具版本

1. 管理员会发给你新版本的 zip
2. 把旧文件夹中的 `.env` 文件复制到新目录（保留 API Key 和 Cookie 配置）
3. 运行新目录下的 `首次安装.bat` 更新依赖
4. 以后双击 `启动工具.bat` 使用新版

### Q: 关闭窗口后还能看到之前的分析结果吗？

可以。所有结果保存在 `output/` 目录下，用记事本打开 `report.md` 即可查看。

## 六、注意事项

- **不要** 把 `.env` 文件分享给他人（包含你的 API Key 和 Cookie）
- **不要** 删除 `venv/` 目录（虚拟环境，删了需要重新安装）
- **不要** 把工具目录移动到带中文空格的路径（虽然大部分情况兼容，但为稳妥建议放桌面或 D 盘根目录）
- **建议** 每周运行一次 `获取抖音Cookie.bat` 保持 Cookie 有效
- **建议** 定期清理 `output/` 目录释放硬盘空间

## 七、技术架构（给技术成员参考）

```
用户输入链接
    ↓
core/downloader.py  — 三引擎降级下载（douyin-downloader → yt-dlp → Playwright）
    ↓
core/audio_extractor.py  — ffmpeg 提取音频
    ↓
core/transcriber.py  — Faster-Whisper 语音转文字 + 字级时间戳
    ↓
core/scene_detector.py  — PySceneDetect 画面切点检测
    ↓
core/segmenter.py  — DeepSeek API 语义分镜
    ↓
core/fuser.py  — 融合对齐（语义边界 + 画面切点 + 时间戳）
    ↓
Gradio Web 界面  — 展示结构化分镜结果
```

每个模块独立可替换，不依赖 UI 框架。

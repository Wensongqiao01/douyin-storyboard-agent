# 抖音 Cookie 获取教程（无需代码基础）

本教程面向剪辑师等非技术用户，约 2 分钟完成。

---

## 第一步：打开抖音网页版并登录

1. 打开 Chrome 浏览器（没有的话装一个：[下载 Chrome](https://www.google.cn/chrome/)）
2. 地址栏输入 `https://www.douyin.com` 并回车
3. 点右上角「登录」，用手机抖音扫码完成登录

> **重要**：登录成功后，你能看到抖音首页的视频流，再继续下一步。

---

## 第二步：打开开发者工具

按键盘上的 **F12** 键（Mac 用户按 **Cmd + Option + I**）。

浏览器右侧或底部会弹出一个面板，不用怕，我们只看一眼。

---

## 第三步：刷新页面

在开发者工具打开的情况下，按 **F5** 刷新页面。

---

## 第四步：找到 Cookie

1. 在开发者工具顶部，点击 **Network**（中文叫"网络"）
   > 如果顶部标签栏被压缩看不到字，把面板拉宽一点
2. 左侧会出现一个请求列表，找到最上面那个 `www.douyin.com`，点击它
3. 右侧会出现详情面板，点击 **Headers** 标签
4. 往下滚动，找到 **Request Headers** 区域
5. 找到 `Cookie:` 这一行
6. 把 Cookie 后面**整段文字**选中复制（很长，可能有几百上千个字符）

```
示意图：
┌─────────────────────────────────────────────────┐
│ Request Headers                                  │
│   ...                                            │
│   Cookie: enter_pc_once=1; uid_tt=abc...  ← 复制这整段  │
│   ...                                            │
└─────────────────────────────────────────────────┘
```

> **技巧**：双击 `Cookie:` 后面的文字可以快速全选，然后 Ctrl+C 复制。

---

## 第五步：发送给管理员

把复制的内容粘贴到微信/钉钉/飞书发给你。

格式大概长这样：
```
enter_pc_once=1; UIFID_TEMP=b684c79658e5909c...; uid_tt=7e7938c636ebc9...; sessionid=afc9594d56a15368...; （后面还有很多）
```

管理员收到后会配置到服务器上。

---

## Cookie 过期了怎么办？

抖音 Cookie 通常有效期 1-3 个月。如果某天分析任务失败，提示 Cookie 相关错误，按上面的教程重新获取一次发给管理员即可。全程不超过 2 分钟。

---

## 常见问题

**Q: 我用的是 Edge 浏览器，可以吗？**
A: 可以。Edge 也能按 F12 打开开发者工具，步骤完全一样。

**Q: 我找不到 Network 标签？**
A: 开发者工具顶部有一排标签，Network 可能在 `>>` 折叠菜单里，点开就能看到。

**Q: 复制到的内容很短，只有几十个字符？**
A: 可能没有登录，或者选错了请求。回到第一步确认登录成功，然后重新操作。

**Q: 我担心隐私，这个 Cookie 能看到我的账号密码吗？**
A: Cookie 不是密码，看不到你的密码。它只是一个登录凭证，用于系统帮你下载你指定的视频。管理员和系统不会用它做任何其他事情。
